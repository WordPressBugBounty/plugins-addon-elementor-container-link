Enhance your Elementor with Addon Elementor Container Plugin by adding custom URL or Link to container, section, inner sections or columns.

== Description ==

Addon Elementor Container Plugin extends the functionality of Elementor by introducing an anchor link field within container, section, inner sections or columns. After installing and activating the Addon Elementor Container Link plugin, navigate to your Elementor editor. When editing a section or container, you will find a new option labeled "Container Link" under the Layout tab. Upon activation, user can easily designate these elements as clickable, enhancing user experience and navigation on Elementor-powered websites. 
This feature enriches the default capabilities of Elementor, providing users with greater control over interactivity and engagement.

== Features ==

- Add link, custom url to make elementor container, section, inner sections and columns clickable.
- Enhance user experience by making containers elements clickable.
- Link 'Open in new window' link Options.
- Dynamic tags
- Compatible with Elementor version 3.29.2 and higher.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/addon-elementor-container-link` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= How do I add a link to an Elementor container?

After installing and activating the Addon Elementor Container Link plugin, navigate to your Elementor editor. When editing a section or container, you will find a new option labeled "Container Link" under the Layout tab. Enter the desired link URL in the provided field, and the container will become clickable on the frontend.

= Can I customize the appearance of the clickable containers? 

Yes, you can customize the appearance of the clickable containers using custom CSS. Since Elementor provides extensive styling options, you can use the built-in features to style the containers according to your preferences.

== Note == 
Our plugin requires the installation of Elementor for full functionality. Access the anchor or custom URL field option in the Elementor edit page interface to unlock enhanced interactive features.
